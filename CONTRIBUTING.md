# Contributing

Thanks for helping improve Credara Contracts.

## Development Workflow

1. Install dependencies with `npm install`.
2. Make focused changes.
3. Add or update tests for contract behavior.
4. Update docs when behavior or assumptions change.
5. Run `npm run compile`, `npm run test`, and `npm run typecheck`.

## Pull Request Guidelines

- Keep pull requests scoped and easy to review.
- Explain security and privacy impacts.
- Do not include private keys, `.env` files, secrets, or sensitive student records.
- Use demo data only.

## Issue Guidelines

For bugs, include clear reproduction steps. For features, explain the use case and privacy impact.
